Theme Selector

By setting the CSS properties --bit-color-0 and --bit-color-1 on the <html> element, you can apply a custom color theme to the entire page.

Get Creative!

    

      CSS Custom Properties can be changed on your page at any time! You can use
      JavaScript, or even CSS selectors to do it! You could apply a :focus style
      to change --bit-color-0 and --bit-color-1 to different colors to make the
      focus state easier to read.
    

    

      Or if you have a super dangerous delete button, you could force the color
      to bed red so it stands out against your normal color scheme. 1-bit UI
      encourages your creativity.
