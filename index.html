1bit-ui.css

Overview

1-bit UI is a CSS library for making retro-inpsired web pages using just two colors. This color palette is completely customizable using CSS Custom Properties (also known as CSS Variables).

1-bit UI covers basic form fields (radio button, checkbox, button, select), links, tables, and cards. Most other basic elements look great alongside these. Evertyhing in 1-bit UI requires the use of CSS classes, so you can feel confident that 1-bit UI won't clash with your existing CSS. 1-bit UI also uses the lowest specificity possible for each rule, so it's less difficult to override things.

      Firefox has the ability to put custom colors on your scrollbars! Don't

Get Creative!

    

      CSS Custom Properties can be changed on your page at any time! You can use
      JavaScript, or even CSS selectors to do it! You could apply a
      :focus style to change
      --bit-color-0 and
      --bit-color-1 to different colors to make
      the focus state easier to read.
    

    

      Or if you have a super dangerous delete button, you could force the color
      to bed red so it stands out against your normal color scheme. 1-bit UI
      encourages your creativity.
    

    


    
Dark Mode

    

      You can use the
      prefers-color-scheme: dark CSS media query
      to set the --bit-color-0 and
      --bit-color-1 variables, allowing your users
      to automatically get Dark Mode with a single block of CSS
    

    
.bit-root {
  --bit-color-0: white;
  --bit-color-1: black;
}

@media (prefers-color-scheme: dark) {
  .bit-root {
    --bit-color-0: white;
    --bit-color-1: black;
  }
}
